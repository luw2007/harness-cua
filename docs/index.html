cua-harness

Agent-friendly Python SDK for desktop automation. Control macOS apps through Accessibility APIs without stealing foreground focus.

GitHub Get Started

Features

30+ Tool Wrappers

click, type_text, press_key, scroll, screenshot, get_window_state, drag, launch_app and more — all through element indices or coordinates.

wait_for()

Poll until a UI condition is met. Composable with any predicate function. Configurable timeout and interval.

AX Tree Diff

Capture two accessibility snapshots and get a structured diff of added, removed, and changed elements.

Macro Record / Replay

Record interaction sequences to JSON trajectories. Replay at configurable speed for regression testing.

Performance Profiler

Automatic per-tool latency tracking. Call counts, avg/min/max stats. Zero-config — wired into every call.

Dry-Run Mode

Toggle mutations off globally. Read-only tools still execute. Perfect for testing agent plans safely.

Install

Requires cua-driver on PATH

pip install cua-harness

Quick Start

from cua_harness import ensure_daemon, launch_app, get_window_state, click, type_text

ensure_daemon()

app = launch_app("com.apple.Safari")
pid = app["pid"]

state = get_window_state(pid, capture_mode="som")
click(pid, element_index=5)
type_text(pid, "hello from cua-harness")
